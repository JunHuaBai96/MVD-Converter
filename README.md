## 元素摩尔体积与密度转换器 (MVD-Converter)

一个用于计算化学元素摩尔体积与密度的交互式 Web 工具。通过选择元素并输入已知的摩尔体积或密度，程序会根据公式自动推算另一个物理量，适合基础化学学习与简单科研辅助。

---

### 功能特性

- **元素选择**：支持多种常见元素（H、He、C、N、O、Fe、Cu、Au、Pb 等），展示名称、符号、原子序数、摩尔质量与常见密度。
- **双向转换**：输入摩尔体积 \(V_m\) 自动计算密度 \(\rho\)，或输入密度反算摩尔体积；只需填写一个值。
- **结果展示**：逐步显示公式与计算过程，高亮最终结果，区分成功与错误提示。
- **友好交互**：单页 HTML，支持回车触发计算，默认选择铁 (Fe)，响应式布局兼容移动端。

---

### 使用公式

- 密度：\(\rho = \dfrac{M}{V_m}\)
- 摩尔体积：\(V_m = \dfrac{M}{\rho}\)

单位约定：摩尔质量 \(M\) 为 g/mol，摩尔体积 \(V_m\) 为 cm³/mol，密度 \(\rho\) 为 g/cm³。

### 快速开始

1. 克隆或下载本仓库。
2. 使用浏览器打开 `MVD-Converter.html`（双击或通过「文件 → 打开」）。
3. 选择元素，输入摩尔体积或密度，点击“计算转换”即可查看结果。

> 本工具无需任何依赖或后端环境，只要现代浏览器即可运行。

### 使用说明

1. 在左侧“元素选择”中选定元素。
2. 在输入框中填写摩尔体积 (cm³/mol) 或密度 (g/cm³) 的任意一个。
3. 点击“计算转换”或按 Enter，查看计算过程与结果。
4. 需要重新输入时点击“重置输入”。
